Schritte zum anlegen einer neuen Mod

-- New Project --

File > New > Projects > Java > Java Projects


-- Change Directories --

Delete src Source Folder

(Right Click) Project > New > Source Folder

"scr/java" , "src/resources"

Check "Update exclusion filters in other source folders to solve nesting"


-- Set Forge to Project --

(Right Click) Project > Build Path > Configure Build Path > Select Project Tab > Add > Select Forge Project


-- Set Run Configurations --

(Right Click) Project > Run As > Run Configurations > New Java Aplication

Client:

Main.class = GradleStart
Programm Arguments = "--username=username --password=password"

Server:

Main.class = GradleStartServer
